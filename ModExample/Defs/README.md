# 注意
此文件夹存放配置项，游戏会自动读取这些配置数据，会修改游戏中的配置

# 教程
excel_data中保存着原始的数据表格，第一行为描述，第二行为数据字段名称及数据类型

json_data中保存着游戏运行时，所使用的数据，则excel_data转化而来

*Mod如何修改这些数据？*
从json_data中拷贝一份数据出来，只保留一条数据，删除多余数据。

修改那条数据的内部，保存到Mod文件夹的Defs中，该Mod被启用后，即可修改游戏运行时的数据。

如果修改数据的Id, 则会新生产一条数据。

*示例*
比如通过stuff.json，分配一个新的id, 添加一条数据，即可新添加一个游戏元素。

通过blueprint.json，可以添加一个新的配方

通过build.json，可添加一个新的建筑。但是建筑要起作用，则需编写相关的代码。
